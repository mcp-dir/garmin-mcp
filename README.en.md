# Garmin Connect (treinos)

### Garmin Connect (treinos) for Claude, Cursor and AI agents

List, fetch, create, edit (replace), delete and schedule structured workouts on Garmin Connect (running, cycling, swimming, multi-sport / triathlon). Uses the unofficial API via Garmin session (email + password); OAuth tokens are stored encrypted to reduce sign-ins.

- 📊 **0 tools**
- 🔒 **Read-only**
- 💬 **Works with any MCP client**: Claude Desktop, Cursor, VS Code, Cline, Continue
- 🔑 **Magic-link login (no password)**
